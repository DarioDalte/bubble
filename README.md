## Getting Started

First, install all the dependencies

```bash
npm install
# or
yarn install
```

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

The website has also been published online using [Vercel](https://vercel.com/) and can be found [here](https://bubble-three.vercel.app/)
